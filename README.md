#  TaskFlow - Task Management Dashboard

A modern and responsive **Task Management Dashboard** built using **React.js**, **Vite**, **JavaScript**, **Tailwind CSS**, and **shadcn/ui**. The application provides mock JWT authentication, task management with CRUD operations, dashboard analytics, dark mode, and a responsive user interface.

---

##  Overview

TaskFlow helps users efficiently manage their daily tasks through a clean and intuitive dashboard.

The application includes:

- Mock JWT Authentication
- Protected Routes
- Dashboard Analytics
- Task Management (CRUD)
- Search & Filter
- Dark/Light Mode
- Responsive Design
- Profile Management

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/rajendra1322/TaskFlow
```

### Navigate to Project Folder

```bash
cd task
```

### Install Dependencies

```bash
npm install
```

---

##  Run the Frontend

```bash
npm run dev
```

Frontend runs at:

```text
http://localhost:5173
```

---

##  Run the Mock Backend

```bash
npm run server
```

JSON Server runs at:

```text
http://localhost:3000
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
VITE_API_URL=http://localhost:3000
```
